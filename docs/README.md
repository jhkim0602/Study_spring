# lect_B Study Docs

이 폴더는 스프링 실습 코드와 분리해서 공부 내용을 Markdown으로 정리하는 공간이다.

목적:

- `src/` 아래 자바 코드에 바로 적용하지 않은 내용을 기록한다.
- Git에 그대로 올릴 수 있게 주차별 폴더로 관리한다.
- 실습 전에 개념을 정리하고, 실습 후에는 결과와 느낀 점을 누적한다.

구조:

- `docs/week02/README.md`
- `docs/week03/README.md`

운영 규칙:

1. 새 주차가 시작되면 `docs/weekXX/README.md`를 만든다.
2. 코드에 아직 적용하지 않은 내용은 먼저 `docs/`에 정리한다.
3. 실제 프로젝트에 적용한 뒤에는 문서 아래에 결과와 수정 파일을 추가로 적는다.
