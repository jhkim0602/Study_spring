# lect_B Study Docs

이 폴더는 스프링 실습 코드와 분리해서 공부 내용을 Markdown으로 정리하는 공간이다.

목적:

- 주차별로 어떤 실습을 했는지와 무엇을 학습했는지 기록한다.
- `src/` 아래 실제 실습 코드와 연결해서 이해할 수 있게 정리한다.
- Git에 그대로 올릴 수 있게 주차별 폴더로 관리한다.
- 실습 전에 개념을 정리하고, 실습 후에는 결과와 느낀 점을 누적한다.

구조:

- `docs/week02/README.md`
- `docs/week03/README.md`

운영 규칙:

1. 새 주차가 시작되면 `docs/weekXX/README.md`를 만든다.
2. 실습 코드는 `src/`에서 관리하고, 학습 내용 정리는 `docs/`에서 관리한다.
3. 실제로 진행한 실습 내용과 배운 점을 주차 문서에 계속 누적한다.

현재 정리 상태:

- `week02`: JSP와 기본 MVC 구조 확인
- `week03`: 스프링 컨테이너, Bean, XML 설정, Java 설정, `ModelAndView`로 객체 전달
